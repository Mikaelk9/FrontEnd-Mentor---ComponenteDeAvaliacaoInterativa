## Visão Geral

### O Desafio

Os usuários devem ser capazes de:

- Ver o layout ideal do aplicativo dependendo do tamanho da tela do dispositivo
- Ver estados de hover para todos os elementos interativos da página
- Selecionar e enviar uma nota de avaliação
- Ver o cartão de "Obrigado" após enviar a avaliação

## Meu Processo

### Construído com

- HTML5 – Estrutura semântica e organização do conteúdo.
- CSS3 – Estilização, design responsivo e layout em grid.
- JavaScript (Vanilla JS) – Manipulação dinâmica dos dados e interatividade dos botões.
- Google Fonts (Overpass) – Fonte personalizada para a tipografia do projeto.
